## Using vue-components

**The library assumes the consuming application uses Vue.js v2.6+.**

#### 1. Install via npm

`npm install @wmde/wikit-vue-components`

#### 2. Import styles

In your style entry point(s), e.g. `_main.scss`, add

`@import '~@wmde/wikit-vue-components/dist/wikit-vue-components.css';`

or in your entry component, e.g `App.vue`, add

`import '@wmde/wikit-vue-components/dist/wikit-vue-components.css';`

#### 3. Import a component

`import { Button } from '@wmde/wikit-vue-components';`

## Using tokens

The Design System tokens are packaged separately from the `vue-components`.
You can use them in any type of front end.

#### 1. Install via npm

`npm install @wmde/wikit-tokens`

#### 2. Import

The tokens are served in three different formats:

`@import '~@wmde/wikit-tokens/dist/_variables_.less';`

`@import '~@wmde/wikit-tokens/dist/_variables_.scss';`

`@import '~@wmde/wikit-tokens/dist/_variables_.css';`
